Die vier Fünfsterne-Fonds von Quincy bieten einen einfachen Weg, um Diversifizierung zu erreichen. Da Sie in Hunderte oder sogar Tausende von Wertpapieren investieren, hilft Ihen eine breite Diversifizierung, die Schwankungen des Finanzmarkets für Ihr Portfolio abzufangen. Wenn Sie die Merkmale jeden dieser Fonds genauer besehen, werden Sie verstehen, warum sie von Morningstar mit fünfe Sternen bewertet wurden.

Der hochverzinsliche Anleihefonds von Quincy
Fonds-Typ: Anleihe - Unternehmen
Ziel: Der hochverzinsliche Anleihefonds von Quincy zielt auf ein hohes im Rechnungsabschnitt anfallendes Einkommen ab; langfristige Kapitalsteigerung ist ein sekundäres Ziel.

Der Quincy-Wachstumsfonds
Fonds-Typ: Wachstum
Ziel: Der Quincy-Wachstumsfond zielt auf langfristigen Kapitalzuwachs ab, mit dem sekundären Ziel von befristetem Einkommen für den Rechnungsabschnitt.

Der Wachstums- und Einkommensfonds von Quincy
Fonds-Typ: Wachstum und Einkommen
Ziel: Dieser Fonds zielt auf einen hohen Gesamtertrag durch die Kombination von Kapitalsteigerung und im Rechnungsabschnitt anfallenden Einkommens ab.

Der Wagnisfinanzierungsfonds von Quincy
Fonds-Typ: Wachstumssektor - Technologie
Ziel: Der Wagnisfinanzierungsfonds von Quincy zielt auf langfristige Kapitalsteigerung ab.
