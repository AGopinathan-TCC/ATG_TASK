ATG 10.0.3



Welcome to ATG 10.0.3. To help you get started, this page introduces some of the keys to using the various ATG products:

  • Accessing the Documentation
  • Starting the MySQL SQL Database
  • Starting ATG 10.0.3 for Evaluation
  • Starting the ATG Control Center
  • Accessing the ATG Dynamo Server Admin

Note: This document covers the following ATG products:

ATG Adaptive Scenario Engine
ATG Portal
ATG Business Commerce
ATG Consumer Commerce
ATG Content Administration

However, since these products can be installed separately, they may not all be present on your system.


Accessing the Documentation

The ATG 10.0.3 manuals are available on the Oracle Web site.

The ATG Control Center (ACC) also has its own built-in help system. (See Starting the ATG Control Center for more information about the ACC.)


Starting the MySQL SQL Database

Because ATG 10.0.3 needs access to a SQL database in order to run, the installation includes an evaluation copy of the MySQL Engine. You can start the MySQL Engine by selecting the Start MySQL Server icon from an ATG 10.0.3 program group.

Note: The MySQL database must be running before you start ATG 10.0.3.


Starting ATG 10.0.3 for Evaluation

The way to start up ATG 10.0.3 differs depending on your application server:

  • If your application server is JBoss Application Server, select the Start ATG 10.0.3 icon from any of the various ATG 10.0.3 program groups on the Windows Start menu.
  • If your application server is Oracle WebLogic Server or IBM WebSphere Application Server, start up your application server normally. This automatically runs a sample ATG application that was built and deployed by the ATG installer.

For more information about starting up ATG 10.0.3, see the ATG Installation and Configuration Guide.



Starting the ATG Control Center

The ATG Control Center (ACC) is ATG 10.0.3's integrated user interface. To start the ACC:

  1. Start ATG 10.0.3 if it isn't already running (see above).

  2. Select the ATG Control Center icon from the ATG 10.0.3 > Tools program group on the Windows Start menu.

  3. Once the ACC has started, you'll see the Connect to Server screen. Enter the name of the host server you want to connect to, the RMI port number, and a valid user name and password. By default, the initial settings for ATG 10.0.3 running locally on your machine are:
User Name: admin
Password: admin
Locale: English (United States)
Host Name: localhost
Port: 8860

For a full description of the various ATG Control Center screens, see the ACC online help system.


Accessing the ATG Dynamo Server Admin

The ATG Dynamo Server Admin provides a basic web interface that you can use to browse the hierarchy of Nucleus components. To access the Admin for ATG 10.0.3 running locally on your machine, select the ATG Dynamo Server Admin icon from the ATG 10.0.3 > Tools program group on the Windows Start menu.

The ATG Dynamo Server Admin is password-protected by default. The initial settings are:

User Name: admin
Password: admin

For more information about the Admin, see the ATG Installation and Configuration Guide.
