This is version 1.0.4 of the Jakarta Implementation of the JSTL (JSP
Standard Tag Library). For more information on JSTL, and the source
code to this implementation, see:

  http://jakarta.apache.org/taglibs/doc/standard-doc/intro.html

You can import standard tag library it in your web application by adding:

    <taglib>
      <atg-taglib-src-dir>
        /DAS/taglib/jstl/1.0
      </atg-taglib-src-dir>
    </taglib>


to your dynamoJ2EESpecifier_1_3.xml, and the using runDarina with the
build option.
